# README

## Установка node_module'й
### сервер
```
npm install
```
### clinet
```
cd client
npm install
cd ../ 
```
## БД и ПОРТ
```
Прежде чем стартовать, следует:
создать .env файл, в котором указываются
- PORT (не обязательно), по дефолту стоит 8081
- MONGO_URL - подключение монго ссылки
```
## start app
```
npm run dev
```
